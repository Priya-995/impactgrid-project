## ЁЯФе Key Features

### ЁЯУМ Case Reporting System
Structured intake form capturing severity, category, location, and resource needs тАФ fed directly into the prioritization engine.

### ЁЯЩЛ Volunteer Registration & Profiles
Volunteers register with skills, availability, and preferred case types. This becomes the input layer for intelligent matching.

### ЁЯФН Explainable Matching Engine *(Core Innovation)*
Unlike black-box systems, ImpactGrid tells you **exactly** why a volunteer was chosen:

```yaml
Volunteer: Priya Sharma
Match Score: 87%

Reasoning:
  тЬЕ Skill Match:       Medical First Aid + Flood Rescue    тЖТ +40pts
  тЬЕ Proximity:         2.1 km from incident location       тЖТ +25pts
  тЬЕ Availability:      Available immediately               тЖТ +25pts
  тЬЕ Category Fit:      Preferred: Medical Emergencies      тЖТ +10pts - 13pts deducted (partial skill gap)

Decision: BEST AVAILABLE MATCH тАФ Dispatching now.
```

> Transparency builds trust. Trust drives faster action.

### ЁЯдЦ AI Operations Insight Engine
Powered by **Grok API**, ImpactGrid generates real-time operational intelligence:
- ЁЯФ┤ Identifies high-priority clusters of cases
- ЁЯУЛ Suggests NGO actions based on active case load
- ЁЯУК Surfaces patterns in crisis data for predictive awareness

### ЁЯФД Real-Time Dashboard
A live coordination view for NGO operators тАФ active cases, volunteer statuses, assignments, and AI recommendations in one unified screen.

## ЁЯЪА Getting Started

### Prerequisites

```bash
node >= 18.x
npm >= 9.x
MongoDB (local or Atlas)
```

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/your-username/impactgrid.git
cd impactgrid

# Install frontend dependencies
cd impactgrid-frontend
npm install

# Install backend dependencies
cd ../server
npm install
```

### Environment Setup

Create a `.env` file in `/server`:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
GROK_API_KEY=your_grok_api_key
```

Create a `.env` file in `/client`:

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

### Run Locally

```bash
# Start backend
cd server
npm run dev

# Start frontend (new terminal)
cd impactgrid-frontend
npm run dev
```

Visit `http://localhost:5173` ЁЯОЙ
